Class FilterMapDivReduce

Clase FilterMapDivReduce que extiende de NumberListProcessor. Esta clase implementa la operación reduce para dividir todos los elementos de un arreglo numérico y proporciona implementaciones opcionales para los métodos hook afterFiltering y afterMapping.
